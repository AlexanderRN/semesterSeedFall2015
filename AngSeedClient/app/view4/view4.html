 Architecture This Seed provides a Java JAX RS Backend and an Angular Frontend in two separate NetBeans Projects
  • All Backend Code should be placed in Backend Project
  • All Frontend Code should be placed in the Frontend Project, that is DO NOT put any content into the "Web Pages" folder of the Backend Project
When you execute the code (right click AngSeedProject and select run) all content of the Frontend Project will be copied into the "Web Pages" folder.
Backend Architecture This should come as no surprise to you. Its a basic NetBeans Web Project with a few REST Services Buildt in + support for Token-based Security
